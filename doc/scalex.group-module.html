create(name='', description='', providerAccountId=-999, periodType='monthly', budgetLevel='org', startTime=0, endTime=0, limits={})

 

Create a budget

Parameters:
  • name (string) - Budget name
  • description (string) - Budget description
  • periodType (string) - Period Type of budget, default value is monthly. Valid values are:
    • monthly
    • weekly
  • budgetLevel (string) - Budget Level, default value is org. Valid values are:
    • org, Organization
    • user, User
    • role, Role
  • startTime - Start time formatted like 2012-12-12-00:00, default is now
  • endTime - End time formatted like 2012-12-12-00:00, default is one year later
  • limits (list) - Soft and Hard Limits in your local currency. Default is 0, get limits use scalex.budget.getLimitBean(budgetLevel), change it's softLimit and hardLimit.
Returns: dict
Budget just created
