getJobs(type='script', object={})

 

Get jobs of a given script Example:

  • getJobs(type='patch') # get applied patch jobs
  • getJobs(type='script', object=script) # get script jobs
Parameters:
  • type (string) - Job type, default is script. Valid values are:
    • script, get script jobs
    • patch, get applied patch jobs
    • update, get applied update jobs
  • object (dict) - If type is script, object is the script returned by scalex.script.getScripts()
Returns: list
List of jobs

Note: Currently support script jobs only.

update(job, name, script, targets, arguments=[], type='script', version=-1, serverGroups=[], scheduleType=0, startTime=0, repeatInterval=60, endTime=0, repeatCount=0)

 

Update a job

Parameters:
  • job (dict) - Job returned by getJobs()
  • name (string) - Job Name
  • script (dict) - Script returned by getScripts()
  • targets - Targets returned by scalex.node.getNodes() or a single node.
  • arguments (list) - Arguments of the script, default is []
  • scheduleType (int) - Schedule type of job, default is 0, valid values are:
    • 0, Run Once
    • 1, Recurring
  • startTime - Start time formatted like 2012-12-12-00:00, default is now
  • repeatInterval - Repeat interval of recurring schedule, default is 60 mins.
  • endTime - End time of recurring schedule, formatted like 2012-12-12-00:00. You must specify this argument if you want to schedule a recurring job and with a repeat interval.
  • repeatCount (int) - Repeat count of a recurring scheduled job.
Returns: dict
Job just updated
